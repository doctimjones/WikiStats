API Access

Your API key is 'an-auth-token'

In order to gather data from the API, you will need to POST your API token in your header call. The most basic call would be:

 curl -X POST -H 'X-Authentication-Token: 'an-auth-token' localhost:8080/hits/

Returns:

{"count": "1","token": "an-auth-token-hit"}

This will return the number of times the API has been accessed and an authorization token.



To get monthly counts for a given search term, you need to provide the query term and the month in the following format:

 curl -X POST -H 'X-Authentication-Token: 9s2xX3RgH7jcHf, Search: Obama, Month: 08' localhost:8080/hits/

This will return the number of page hits that a wikipedia entry related to the search term Obama saw in the entire month of October:

{ "barack_obama": "325,671", "barack_obama_sr": "6,942" }
